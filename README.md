# 🏠 House Price Prediction (Python Project)

## 📌 Objective
To analyze housing data and build a machine learning model to predict house sale prices based on various features such as location, property characteristics, and structural attributes.

## 🛠 Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook

## 📂 Dataset Features
- Lot Area
- Basement Area
- Year Built
- Year Remodeled
- Overall Condition
- Zoning & Location
- Sale Price (Target Variable)

## 🔍 Project Workflow

### 1. Data Exploration (EDA)
- Analyzed distributions of numerical variables
- Identified missing values
- Visualized relationships using:
  - Histograms
  - Scatter plots
  - Box plots
  - Heatmaps

### 2. Data Cleaning
- Handled missing values using mean/median/mode
- Removed or treated outliers
- Ensured data consistency

### 3. Feature Engineering
- Created new features (Property Age)
- Applied log transformation to SalePrice
- Encoded categorical variables
- Scaled numerical data

### 4. Model Building
- Split data into training and testing sets
- Applied regression models
- Evaluated using:
  - MAE
  - MSE
  - RMSE
  - R² Score

## 📊 Key Insights
- Newer and renovated houses have higher prices
- Basement area and living space strongly impact pricing
- Location significantly influences property value
- Overall condition directly affects sale price
